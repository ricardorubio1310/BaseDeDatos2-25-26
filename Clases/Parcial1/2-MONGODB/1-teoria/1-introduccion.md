# Clase 5 – Teoría de bases de datos documentales y fundamentos de MongoDB

### Objetivo de la clase

Analizar el modelo documental desde una perspectiva conceptual y técnica, comprendiendo cómo se estructura la información y por qué MongoDB es una implementación relevante de este paradigma.

### Repaso: ¿Qué es una base de datos documental?

Una base documental almacena datos en documentos, generalmente en formato JSON o BSON.

Cada documento representa una entidad completa y puede tener estructura propia.

No existe un esquema rígido obligatorio.
